🐍 Python Automation Tools

This repository contains simple and useful Python automation scripts.

📂 Projects
1️⃣ File Organizer
Automatically organizes files into folders based on type
Example:
.txt → text_files
.jpg → photos
.mp3 → songs
2️⃣ Smart Backup System
Creates backup of files and folders
Adds current date to backup name
Prevents overwriting existing backups
⚙️ Technologies Used
Python
pathlib
shutil
datetime
🚀 How to Run
Download or clone this repository
Open terminal in the folder
Run:
python file_organizer.py

or

python smart_backup.py
